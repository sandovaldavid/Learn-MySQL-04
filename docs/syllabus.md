# Syllabus del curso **SQL con MySQL: Proyecto final** de Alura Latam

**Carga horaria:** 8 horas  
**Nivel:** Proyecto práctico avanzado (requiere experiencia previa en SQL y MySQL)

---

### Syllabus / Temario por secciones

**1. Proyectando la base de datos**

-   Presentación
-   Preparando el ambiente
-   Creando la base de datos y tablas
-   Insertando registros (individuales y en lote)
-   Repaso práctico de lo aprendido

**2. Función aleatorio**

-   Uso de la función `RAND()` en MySQL
-   Generación de números aleatorios
-   Creación de funciones personalizadas
-   Aplicaciones de la aleatoriedad en bases de datos

**3. Función cliente aleatorio**

-   Obtención de números enteros aleatorios y uso de `LIMIT`
-   Implementación de funciones para selección aleatoria de cliente, producto y vendedor
-   Práctica y repaso de funciones creadas

**4. Generando ventas y problema con PK (Primary Key)**

-   Implementación de funciones y Stored Procedures para la simulación de ventas
-   Solución de problemas comunes con claves primarias
-   Ciclos iterativos y generación automatizada de registros de ventas

**5. Stored Procedures y TRIGGERS**

-   Creación y mejora de procedimientos almacenados (Stored Procedures)
-   Implementación y perfeccionamiento de TRIGGERS
-   Aplicación de procedimientos almacenados en triggers
-   Formato y consulta de facturación
-   Proyecto final y conclusiones

---

**Prerrequisitos sugeridos:**

-   Haber tomado los cursos de “Introducción a SQL con MySQL” y “Consultas SQL avanzando en MySQL”
-   Manejo básico y avanzado de comandos de SQL

---

**Resultados de aprendizaje:**

-   Desarrollo de un proyecto práctico usando MySQL
-   Creación y manipulación de una base de datos con entidades y relaciones
-   Uso avanzado de funciones, procedimientos almacenados y triggers
-   Simulación y gestión de datos realistas en MySQL mediante técnicas automatizadas y aleatorias
